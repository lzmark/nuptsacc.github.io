      为培养大学生职业素养和创新素质，了解当前IT行业领先的技术和企业真实的需求，提升实践能力。2016年8月25日，计算机学院、软件学院一行6人赴炫彩互动网络科技有限公司参观实践，并就游戏制作、移动端SDK产品开发、前端技能等开展主题交流学习.

在这里，有人跟你组队参加技术比赛，也有人跟你谈理想谈人生；有人跟你抱团做项目，也有人跟你一起捣鼓硬件。

web前端

Web front

Web组，致力于网页设计，是网站的颜值担当。搭网页，建博客，一起吹比。我们不需要太多的基础，哪怕你们只会喊666，我们也欢迎你们的加入。

后台开发

back-end develop

python后端开发组会领着大家学习简单的爬虫知识，web开发方面的知识。 在这里你能够自由的交流、学习，很快的体验到编程的乐趣及其给予你的便捷。

应用开发

application development

本组主要专注于对于开发小型的应用软件,分为桌面端和移动端,主要涉及的语言有C#、java、c++、c等，其中以java为主要的研习语言。当然，除此之外，本组也热心于任何与计算机技术的了解和学习。小组主要是一个互相认识，交流并在谈笑之中学习的地方，致力于营造轻松热闹的学习环境。欢迎各位大佬萌新的加入！

网络安全

Internet security

安全组是一个以Web安全与二进制安全为主要方向的小组，主要的活动内容为参加各类CTF竞赛以及实战演练(针对特定系统进行渗透测试，针对特定app进行风险测试等等)

嵌入式开发

embedded development

几个Maker的自嗨之地，智能车，无人机，3d打印，物联网，智能+，还 有你无限大的想象。坚守创新，持续实践，乐于分享，我们来陪你创新，并努力将自己的创意变为现实。

办公室

Office

秘书部是科协办公部门与技术部门之间的桥梁。负责科协所有大型活动的组织和参与，以及做好会议的通知与记录。我们这个部门没有任何门槛，只要你虚心肯学，既能锻炼你活动组织的能力，提升综合素质，又能随时旁听技术部门的授课，向大佬学习。我们有时还会与其他技术部门合作，完成运维工作，在收获友情的同时又能学到你感兴趣的专业知识。

新媒体

new media

新媒体是科协办公室的重要组成部分，承担活动策划宣传、新闻稿撰写、微信推送等工作，是保障科协稳定运行的重要力量。不同于科协的技术部门，新媒体是学术和后勤服务于一体的部门。统筹规划，炫酷海报，纪实新闻，PS与office,激情与汗水，收获每一次比赛活动成功举办的喜悦，实践每一种现实与理想的交锋。
